# **App Name**: DeporteY Aventura Next

## Core Features:

- Content Migration: Migrate existing content, including posts and images, from the current WordPress site deporteyaventura.es to the new Next.js application.
- Product Listings: Create and manage product listings with details, images, and pricing. Support integration with an e-commerce platform (such as Shopify or Snipcart) if desired.
- Blog Section: Implement a blog section with the ability to create, edit, and publish articles related to sports and adventure activities.
- Category Filters: Enable users to filter products and blog posts by category, tags, and other relevant criteria.
- Search Functionality: Implement a search bar to allow users to quickly find specific products, articles, or information on the site.
- Contact Form: Add a contact form for users to send inquiries or feedback directly from the website.
- AI Content Summarizer: Implement an AI tool that summarizes articles or product descriptions, allowing users to quickly grasp the main points. Display summaries inline with options to view full content.

## Style Guidelines:

- Primary color: Forest green (#386641) to convey nature, adventure, and trustworthiness.
- Background color: Very light desaturated green (#F2F4F3).
- Accent color: Mustard (#A68A31), for contrast and calls to action
- Body font: 'PT Sans', a humanist sans-serif for readability and a modern feel.
- Headline font: 'Space Grotesk', a proportional sans-serif font that provides a computerized and scientific look.
- Use clean and modern icons to represent categories, actions, and other key elements. Consider a line-based style for consistency.
- Employ a responsive grid-based layout to ensure the website is accessible and visually appealing on all devices.
- Incorporate subtle animations on page transitions, button hovers, and content loading to enhance the user experience without being distracting.
